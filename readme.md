# AWS Projects Repository

Welcome to my AWS Projects Repository! This is where I upload and document various projects, experiments, and learning exercises related to Amazon Web Services (AWS).

## About This Repository

This repository serves as a collection of small to medium-sized projects that I've worked on while exploring different AWS services and cloud computing concepts. 

## Projects

Here's a list of projects currently included in this repository:

1. **SSM Endpoints**
   - This project deploys a CloudFormation template that sets up the endpoints required for Systems Manager access to an EC2 instance without internet access.

2. **vSRX CDK**
   - This is a sanitized version of a project I did to deploy a Juniper vSRX firewall appliance along with associated EIPs, subnets, route tables, etc.

3. **Cloud Resume Project**
   - The Cloud Resume Challenge is a hands-on project designed to help bridge the gap from cloud certification to cloud job. It incorporates many of the skills that real cloud and DevOps engineers use in their daily work.  
     For more info, visit [cloudresumechallenge.dev](https://cloudresumechallenge.dev/)
